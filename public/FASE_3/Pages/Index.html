Selectores CSS


Los selectores CSS son herramientas que se utilizan para seleccionar elementos HTML y aplicar estilos específicos a ellos.
Son una parte fundamental de la sintaxis CSS, ya que permiten seleccionar elementos específicos de una página web para aplicar estilos.

A continuación, se presentan algunos de los selectores más comunes y su explicación:

Selector universal (*):

Selecciona todos los elementos de la pagina web.

Selector de tipo (p, h1, h2, etc.):

selecciona todos los elementos del tipo especificado.
Por ejemplo, el selector "p" seleccionaría todos los párrafos de la página web.

Selector de clase (.nombre-de-clase):

Selecciona todos los elementos que tengan la clase especificada.
Por ejemplo, el selector ".azul" seleccionaría todos los elementos que tengan la clase "azul".

Selector de ID (#nombre-de-id):

Selecciona el elemento con el ID especificado.
Por ejemplo, el selector "#encabezado" seleccionaría el elemento con el ID "encabezado".

Selector de descendiente (elemento espacio elemento):

Selecciona todos los elementos secundarios que son descendientes del primer elemento especificado.
Por ejemplo, el selector "div p" seleccionaría todos los párrafos que sean descendientes de una etiqueta "div".

Selector de hijo directo (elemento > elemento):

Selecciona todos los elementos que son hijos directos del primer elemento especificado.
Por ejemplo, el selector "ul > li" seleccionaría todos los elementos "li" que son hijos directos de una lista desordenada "ul".

Selector de atributo ([atributo="valor"]):

Selecciona todos los elementos que tengan el atributo especificado con el valor especificado.
Por ejemplo, el selector "a[href="https://www.google.com"]" seleccionaría todos los enlaces que tengan el atributo "href" con el valor "https://www.google.com".

Selector de pseudoclase (a:hover):

Selecciona un elemento en función de su estado o interacción con el usuario.
Por ejemplo, el selector "a:hover" seleccionaría todos los enlaces cuando el usuario pasa el cursor sobre ellos.

Selector de pseudoelemento (p::first-letter):

Selecciona una parte específica de un elemento.
Por ejemplo, el selector "p::first-letter" seleccionaría la primera letra de todos los párrafos de la página web
